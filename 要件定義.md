# Studio DO — ウェブサイトリデザイン 要件定義

> 作成日: 2026-03-23

---

## 1. プロジェクト概要

| 項目         | 内容                                                                                       |
| ------------ | ------------------------------------------------------------------------------------------ |
| クライアント | Studio DO（ダンススタジオ）                                                                |
| 所在地       | 埼玉県さいたま市北区櫛引町2-305-1 櫛引ビル4F                                               |
| 目的         | 既存サイトのリデザイン。プロフェッショナルな印象を強化しつつ、体験レッスンへの導線を最適化 |
| 担当         | 株式会社ドウガク（Mark）                                                                   |

---

## 2. 現状サイトの課題（推定）

- コンセプト・世界観はあるが、デザインの洗練度がスタジオの実力に見合っていない
- インストラクター陣の経歴が非常に豪華（D.LEAGUE、国際賞多数）だが、その凄さが視覚的に伝わりにくい
- CTAが散在しており、体験レッスン申込への導線が弱い
- 情報の優先順位が不明確（News、Concept、Classが同列に見える）

---

## 3. リデザインの方向性

### コンセプト

**「一流の講師陣がいる、大宮の本格ダンススタジオ」** を第一印象で伝える

### トーン & ムード

- **プロフェッショナル**: 講師の実績が自然と目に入るデザイン
- **温かみ**: 「ご縁を繋ぐ」というコンセプトに合った、敷居の低さも感じられるバランス
- **動き**: ダンススタジオらしいダイナミックさ（映像・アニメーション）

---

## 4. サイト構成（ページ構成案）

### シングルページ or マルチページ

→ **推奨: シングルページ（LP型）+ 必要に応じてサブページ**

- メインはスクロール型の1ページ完結
- インストラクター詳細・料金プランは必要に応じて別ページ化

### セクション構成

| #   | セクション             | 目的                                                     | 優先度 |
| --- | ---------------------- | -------------------------------------------------------- | ------ |
| 1   | **ヒーロー**           | ファーストビューで世界観を伝える。動画背景 or 高品質写真 | 最高   |
| 2   | **CTA帯**              | 体験レッスン無料の訴求 + 申込ボタン                      | 最高   |
| 3   | **コンセプト**         | 「踊りを通じてご縁を繋ぐ」の世界観を伝える               | 高     |
| 4   | **クラス紹介**         | バレエ・コンテンポラリー・K-POPなどをカード形式で        | 高     |
| 5   | **インストラクター**   | 写真＋経歴ハイライト。実績の凄さを視覚的に伝える         | 最高   |
| 6   | **体験レッスンの流れ** | 初めての人の不安を解消するステップ説明                   | 高     |
| 7   | **スケジュール**       | 週間スケジュール（埋め込み or テーブル）                 | 中     |
| 8   | **料金プラン**         | シンプルな料金表                                         | 中     |
| 9   | **アクセス**           | 地図＋住所＋最寄り駅からの行き方                         | 中     |
| 10  | **お知らせ**           | News（最新3件程度）                                      | 低     |
| 11  | **お問い合わせ / CTA** | 最終CTA＋フォームまたはLINE誘導                          | 最高   |

## 5. 重要コンテンツの扱い

### インストラクター（最重要差別化要素）

現状の講師陣は国内トップクラスの実績を持つ。これがStudio DOの最大の武器。

**表現方針:**

- 各講師のプロフェッショナルな写真を大きく使う
- 受賞歴・活動歴を「実績バッジ」や「ハイライト」で視覚化
- 「D.LEAGUE所属」「コンドルズメンバー」など一般にも伝わるキーワードを目立たせる
- 詳細経歴はアコーディオンまたは詳細ページで展開

**講師一覧:**

1. **黒須育海** — コンドルズメンバー、ダンスカンパニーブッシュマン主宰、国内外受賞多数
2. **井田亜彩実** — D.LEAGUE「LIFULL ALT-RHYTHM」所属経験、イスラエル留学、国際受賞多数
3. **篠原陽** — Mr.Children MV出演、東京デフリンピック2025パフォーマー
4. **HANA** — ヨガインストラクター

### コンセプト「踊りを通じてご縁を繋ぐ」

- 抽象的なスローガンで終わらせない
- 実際のレッスン風景・生徒の声・コミュニティ感が伝わるビジュアルで補強

### 体験レッスン（CVポイント）

- **初回無料**を最大限に訴求
- ヒーロー直下、中間CTA、最終CTAの3箇所に配置
- LINE公式アカウントへの誘導も併設

## 6. デザイン要件

| 項目           | 仕様                                                                 |
| -------------- | -------------------------------------------------------------------- |
| レスポンシブ   | モバイルファースト（来訪者の大半がスマホ想定）                       |
| カラー         | ダークベース（黒・チャコール）＋アクセントカラー（要相談）           |
| フォント       | 日本語: Noto Sans JP or Zen Kaku Gothic / 英語: モダンなサンセリフ   |
| 画像           | 高品質なスタジオ写真・講師写真が必須（素材提供をクライアントに依頼） |
| 動画           | ヒーローセクションにレッスン風景のショート動画（自動再生・ミュート） |
| アニメーション | スクロール連動のフェードイン。過度にしない                           |

---

## 7. 技術要件

| 項目             | 方針                                            |
| ---------------- | ----------------------------------------------- |
| 実装方式         | 静的HTML/CSS/JS（シンプル・高速）               |
| ホスティング     | GitHub Pages or Vercel（要相談）                |
| ドメイン         | クライアント既存ドメインを使用（要確認）        |
| SEO              | 基本的なmeta設定、構造化データ（LocalBusiness） |
| パフォーマンス   | Lighthouse 90+を目標                            |
| アクセシビリティ | WCAG 2.1 AA準拠を目標                           |
